# Generador de paleta de colores interactivo

Este es una aplicación web simple que genera paletas de colores de manera aleatoria,
tiene el propósito de ayudar al usuario a encontrar los colores a utilizar en sus proyectos
y además genera los códigos de los mismos en dos formatos, HSL y HEX. 

# La aplicación hace escencialmente lo siguiente:

- Permite generar de manera aleatoria una paleta de seis, ocho o nueve colores
- Permite elegir el tipo de código del color que el usuario desea, entre HSL y HEX
- Permite copiar el código del color al hacer click sobre el mismo.

# Lenguajes utilizados :
- HTML 5
- CSS3
- JavaScript

# Cómo usarlo :
1- Selecciona la cantidad de colores
2- Selecciona el formato de código de color
3- Genera la paleta
4- Haz click en el color que te guste
5- El código se copiará automáticamente (saldrá un aviso de "copiado")

# Características
- Generación aleatoria de paletas de colores
- Selección de tipo de código entre HSL y HEX
- Copiado rápido de códigos
- Interfaz simple e intuitiva
- Diseño responsive
